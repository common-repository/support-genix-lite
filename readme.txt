== Support Genix Lite Features ==

**Unlimited Tickets:** Manage as many tickets as you need, there are no limitations on the number of tickets.

**Unlimited Agents:**  Add as many agents as you need to help manage your tickets.

**Unlimited Customers:** There is no limit on the number of customers.

**Custom Ticket Fields:** Customize the fields of your tickets to get the information you need.

**Agent Roles:** Assign different roles to your agents to give them different permissions.

**Ticket Categories:** Create different ticket categories to organize your tickets.

**File Attachments:** Allow customers to attach files to their tickets. You can define the type of allowed files.

**Filters and Sorting:** Use the filters and sorting options to manage your tickets.

**Email Notifications:** Receive/Send email notifications when a new ticket is created or updated. You can customize email templates for Admin/Agent and customers.

**Ticket Status:** Change the status of tickets to keep track of them. Available statuses are - New, In Progress, Re-Open, and Active.

**Ticket Notes:** Add notes to tickets for internal use.

**Tickets on Behalf of Users:** Agents can create tickets on behalf of users.

**Transfer Ticket:** Agents can transfer tickets to other agents.

**Close Ticket:** Close tickets when they are no longer needed.

**Re-Open Tickets:** Re-open closed tickets if needed.

**Ticket Logs:** View the history of a ticket to see who made changes and when.

**Envato Integrations:** Support Genix integrates with popular marketplace like Envato marketplace / ThemeForest / CodeCanyon to verify purchase code.

**Saved Replies / Canned Responses:**  Save replies to use them later. This is helpful when you need to give the same reply to multiple tickets.

**Recaptcha:** Add Google reCAPTCHA to your ticket form to prevent spam.

**Fully Responsive:** The plugin is fully responsive and looks great on all devices.

**Search:** Use the search feature to find tickets quickly and easily.

**No. of replies:**  You can see the total number of replies in a ticket.

**Need Reply Ticket Labels:** To quickly identify which tickets need a reply.

**Order by:** Choose how you want to order your tickets - by date, status, etc.

**Responsive Design:** This WordPress ticket plugin is fully responsive and looks great on all devices.

**Translation Ready:** This WordPress Support plugin is translation ready and can be translated into any language.

**WPML Compatible:** This WordPress ticketing plugin is fully compatible with WPML, allowing you to use it in multiple languages.

**Submit Ticket Form:** The WordPress support ticket plugin includes a ticket form that customers can use to submit tickets or create support tickets.

**Ticket List:** This customer service plugin includes a ticket list where you can see all the tickets.

**Agent Ticket List:** This ticket systems plugin includes an agent ticket list where agents can see all the tickets they are assigned to.

**Customer Ticket List:** This WordPress helpdesk ticket plugin includes a customer ticket list where customers can see all the tickets they have submitted.

**Ticket Details:** This help desk plugin includes a ticket details page where you can see all the information about a ticket.
